## 🎯 Common Tasks

### Task: View your portfolio locally
**Time: 30 seconds**
```bash
cd /home/ezzhar/portfolio
npm run dev
# Opens http://localhost:3001
```

### Task: Change primary color
**Time: 2 minutes**
1. Open `tailwind.config.js`
2. Find `primary` color section
3. Change hex codes
4. Save and refresh browser

### Task: Update project description
**Time: 3 minutes**
1. Open `src/sections/Projects.jsx`
2. Find the project in the array
3. Update description text
4. Save and see changes live

### Task: Add more projects
**Time: 5 minutes**
1. Open `src/sections/Projects.jsx`
2. Add new object to `projects` array
3. Fill in all required fields
4. Save and it renders automatically

### Task: Deploy to Vercel
**Time: 5 minutes**
1. Push code to GitHub
2. Go to vercel.com
3. Click "New Project"
4. Select your repository
5. Click "Deploy"
